# OctoAcme Project Management Docs

Welcome to the OctoAcme project management documentation! This repository contains standardized, living documentation of how projects are run, tracked, and improved at OctoAcme.

## Project Management Processes Summary
OctoAcme uses a structured approach to project delivery:
- **Customer-first:** Prioritize customer value and usability.
- **Iterative delivery:** Deliver small, testable increments.
- **Clear ownership:** Defined project and product leadership.
- **Data-informed:** Make decisions based on evidence and measurement.
- **Team Rhythm:** Regular standups, demos, and syncs for progress and feedback.
- **Quality focus:** Testing, checklist-driven workflows, and risk tracking.
- **Continuous improvement:** Capture learnings and evolve processes after each project or sprint.
